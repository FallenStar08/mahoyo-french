INSTALLATION DU PATCH DE MAHOU TSUKAI NO YORU :

	- Trouvez votre dossier de sauvegarde :
		. Soit il se trouve dans le dossier du jeu dans le dossier "savedata"
		. Soit il se trouve dans vos documents : C:\Users\****\Documents\TYPE-MOON\魔法使いの夜

	- Créez-y un dossier "patches" s'il n'existe pas et mettez le fichier "patch1.xp3" à l'intérieur de celui-ci.

ET VOILA, VOTRE JEU EST PRÊT !

Vous pouvez lancer le jeu via l'exécutable sur windows ou via wine sur Linux.

Linux :
env WINEPREFIX= »/home/username/.wine » LANG=ja_JP.utf-8 wine C:\\windows\\command\\start.exe /Unix /home/username/.wine/dosdevices/e:/TYPE-MOON/魔法使いの夜/魔法使いの夜.exe
